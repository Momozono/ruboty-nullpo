# Ruboty::Nullpo
An ruboty handler to request ｶﾞｯ.

## Usage
```ruby
# Gemfile
gem 'ruboty-nullpo', github: 'mallowlabs/ruboty-nullpo'
```

```
@ruboty nullpo -> ｶﾞｯ.
@ruboty ぬるぽ -> ｶﾞｯ.
@ruboty ヌルポ -> ｶﾞｯ.
```
